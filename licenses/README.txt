Fiix CMMS client for Java
==========================================

Welcome to the Fiix CMMS client for Java project.


Dependencies
------------

Fiix CMMS client requires Java 1.6 compatible runtime and
depends on the following external libraries:

* Apache Commons Codec
* Apache Commons Logging
* Apache Jakarta HttpClient
* Jackson JSON processor
